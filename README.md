# Meu Catálogo

Este é o **Meu Catálogo**, um sistema web que permite gerenciar e visualizar produtos de forma simples e organizada. Com ele, você pode cadastrar novos produtos, editar informações existentes e visualizar todos os itens cadastrados em uma interface limpa e intuitiva.

## Tecnologias Utilizadas

- **HTML**: Estrutura das páginas.
- **CSS**: Estilo e layout responsivo.
- **JavaScript**: Funcionalidades interativas do sistema.
- **Supabase**: Banco de dados em tempo real para armazenar e gerenciar os produtos.
